# decisions

Architecture Decision Records (ADRs).

## Conventions

- One ADR per file: `NNNN-short-kebab-case-title.md`.
- Status: `Proposed`, `Accepted`, `Superseded by NNNN`, or `Deprecated`.
- Sections: Context, Decision, Consequences, Alternatives Considered.
- Link related ADRs if applicable.

ADRs are append-only. To revise a decision, write a new ADR and mark the old one as superseded.
